PRODUCTOS

Babero

Baberos de alimentación impermeables con patrón de dibujos animados. Es ajustable, fácil de poner y no es necesario lavar, además cubre bien la ropa. Ideal para la temporada de primavera-verano.

Cubiertos

Cubiertos de acero inoxidable, fácil de manipular, diferentes colores, el mango es de silicona. Fácil de lavar y es adecuado para niños pequeños.

Manta de muselina

Mantas para bebé con diferentes estampados, el material con el que está hecha: algodón y fibra de bambú, es ligera y fresca. Tamaño: 120x120. Diferentes modelos

Sombrero

Sombrero tipo pescador dibujo de oso. Diferentes tallas y colores. Material:algodón y nylon. Unisex, ideal para cualquier época del año.

Vaso de silicona

Es una tazas de alimentación de silicona, grado alimenticio, también puedes almacenar los aperitivos, sin BPA.Ideal para bebés. Capacidad: 200 ml. Diferentes colores
